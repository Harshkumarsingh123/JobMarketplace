# Full Stack Web Application

## Overview
This is a full stack application built with:
- Spring Boot (Backend)
- React + Vite (Frontend)

## Tech Stack
- Java, Spring Boot, Spring Security, JWT
- React.js, Vite
- MySQL
- REST APIs

## Project Structure
backend   → Spring Boot backend  
frontend  → React Vite frontend  

## Run Backend
cd backend
mvn spring-boot:run

## Run Frontend
cd frontend
npm install
npm run dev
